# 🌊The Deep
The Deep is a 2D side-view underwater exploration game. You play as a lone researcher dispatched to Kepler, a newly discovered and entirely aquatic alien planet. Sent with limited resources, you must dive into the unknown, research alien marine life, harvest resources, and upgrade your gear to survive the crushing depths and hostile predators.

---

## Goal

The final objective of this game is to research the entire alien ecosystem of the planet Kepler. You play as Juan, an explorer who has to upgrade his equipment and survive the depths, while he fills his research database to finally complete his mission and be able to return home.

---

## Prerequisites

* **Python 3.1 or higher**

---

## Installation & Setup

1. **Extract** the project files to your local directory.
2. **Install Dependencies**: Use the provided `requirements.txt` file to install Pygame and any other necessary libraries:
```bash
pip install -r requirements.txt

```


3. **Launch the Game**: Run the following command from the root directory:
```bash
python main.py

```

## 🎮 Controls

The controls change depending on whether you are at the surface base or exploring the ocean.

### In the Homebase

| Action | Key |
| --- | --- |
| **Move Side to Side** | `A` / `D` |
| **Interact with Stations** | `E` |
| **Pause Game** | `ESC` |
| **Debug Mode** | `F5` |
| **Toggle Fullscreen** | `F6` |

### Underwater

| Action | Input |
| --- | --- |
| **Movement** | `W`, `A`, `S`, `D` |
| **Sprint** | `Spacebar` |
| **Aim** | `Mouse Movement` |
| **Use Equipped Item** | `Left Click` |
| **Cycle Holdables** | `Tab` |
| **Interact / Open Inventory** | `E` |
| **Debug Mode** | `F5` |
| **Toggle Fullscreen** | `F6` |

## Gameplay & Progression

1. **Survival & Combat**: Start by hunting creatures by using your weapon and approaching them after they are neutralized to harvest them. 

2. **The Economy**: Collected creatures can be traded for money at the Homebase.

3. **Upgrades**: Use your earnings to upgrade your gear and be able to withstand deeper pressures and face off greater enemies.

4. **Exploration**: When you go deeper you will find more unique and valuable creatures. On the other side, these depths pose a significantly greater problem for survival.

5. **Goal**: Research every specie of the alien ecosystem to be able to return home to Earth.
